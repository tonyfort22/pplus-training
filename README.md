# PPLUS Training

Mobile-first training platform for hockey athletes and coaches.

## Stack
- Expo / React Native
- Next.js
- Supabase / Postgres
- TypeScript
- Zod
- TanStack Query
- Zustand

## Initial focus
- clean training domain model
- workout execution flow
- prescribed vs actual tracking
- fatigue and progress analytics
